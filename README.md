# IT1244 Final Project

This repository contains code and notebooks for the IT1244 Final Project, focused on machine learning classification for cancer detection and health screening. The project includes data preprocessing, model training, and evaluation using various algorithms such as Decision Trees, K-Nearest Neighbors, Logistic Regression, and Gaussian Naive Bayes.

## Structure
- Jupyter notebooks for model development and analysis
- Python scripts for data preprocessing
- CSV files for training and testing datasets

## Usage
1. Clone the repository.
2. Open the notebooks in Jupyter or VS Code.
3. Run the preprocessing and model training cells.
4. Evaluate results and visualize outputs.
